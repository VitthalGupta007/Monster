About VitthalX Studio

VitthalX Studio is a solo-led independent game studio focused on Android. We combine survivor combat, meta progression, and store-ready production — ads, IAP, Play Games services, and honest privacy disclosures — without bloated scope.

VX Monster is our first commercial title: package com.vitthalxstudios.monster, distributed on Google Play and available for review builds on this site.

What we value

  • Readable mobile UI and fair touch targets
  • One-more-run game loops with real mastery
  • Transparent data practices for players and platforms
  • Ship, learn, iterate — not endless pre-production
